⌨️ Comandos de Consola

Abre la consola del navegador (F12) y ejecuta estos comandos para explorar el sistema:

// Ver usuario actual
sgcAuth.getCurrentUser()
// Obtener todos los documentos
sgcDB.getAllDocuments()
// Obtener todos los usuarios
sgcDB.getAllUsers()
// Ver logs de auditoría
sgcDB.getAuditLogs()
// Crear documento manual
sgcDB.createDocument({ name: "Test", code: "TEST-001", type: "pdf", department: "Prueba" })
