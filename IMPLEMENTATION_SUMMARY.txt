================================================================================
                         🔗 API ENDPOINTS NUEVOS
================================================================================

1. GET /api/admin/users/stats
   Descripción: Obtener estadísticas de usuarios
   Retorna: { total, admins, managers, active }
   Acceso: Solo super_admin

2. DELETE /api/users/{uid}
   Descripción: Eliminar usuario
   Parámetro: uid (ID del usuario)
   Validación: No permite eliminar super_admin
   Acceso: Solo super_admin

3. PATCH /api/users/{uid}/toggle-status
   Descripción: Activar/desactivar usuario
   Parámetro: uid (ID del usuario)
   Retorna: { status: "toggled", active: bool }
   Acceso: Solo super_admin

4. GET /api/admin/audit
   Descripción: Obtener registro de auditoría
   Retorna: Últimas 100 acciones registradas
   Acceso: Solo super_admin
