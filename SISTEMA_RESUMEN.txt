## 🧪 Cómo Probar Cada Operación

### USUARIOS
```
1. CREATE:    Click "+Add User" → Completa form → Save
2. READ:      Abre la página → Ve tabla con usuarios
3. UPDATE:    Click "Edit" → Cambia datos → Save
4. DELETE:    Click "Delete" → Confirma → OK
5. TOGGLE:    Click icono switch → Cambia estado
6. SEARCH:    Escribe en search box → Filtra en tiempo real
7. FILTER:    Selecciona role/dept → Filtra
8. EXPORT:    Click "Export CSV" → Descarga archivo
```

### DOCUMENTOS
```
1. CREATE:    Click "+Add Document" → Completa form → Save
2. READ:      Abre página → Ve tabla + stats
3. RENAME:    Click "Rename" → Nuevo nombre → Save
4. STATUS:    Click "Status" → Selecciona estado → OK
5. VIEW:      Click "View" → Popup con detalles
6. DOWNLOAD:  Click "Download" → Simula descarga
7. DELETE:    Click "Delete" → Confirma → OK
8. SEARCH:    Escribe en search → Filtra
9. FILTER:    Status + Department → Filtra combinado
10. EXPORT:   Click "Export CSV" → Descarga archivo
```
