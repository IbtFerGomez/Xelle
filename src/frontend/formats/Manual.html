1. Semáforo de Estatus de Lotes (Identificación Física)

Todo material en almacén o laboratorio debe portar una de las siguientes etiquetas según su fase:

Liberado

2. Gestión de Residuos (RPBI - NOM-087)

  • ET-RPBI-01 (ROJO): Punzocortantes, cultivos, cepas y residuos no anatómicos[cite: 3, 4].
  • ET-RPBI-02 (AMARILLO): Placentas y residuos patológicos (tejidos anatómicos)[cite: 1, 3, 4].

3. Flujo Documental y Puntos Críticos de Control (PCC)

Fase del Proceso Formato Obligatorio Punto Crítico de Control (PCC)
1. Recepción FO-LC-17 / 17B Integridad de embalaje y temperatura (2°C a 8°C).
2. Procesamiento FO-LC-18 / 18B Peso neto, apariencia visual y asepsia de cabina.
Muestreo FO-LC-34 [cite: 3] Toma estratificada y registro de cantidad volumétrica[cite: 3, 4].
3. Calidad FO-LC-19 / 19B Serología, esterilidad y endotoxinas.
Liberación FO-LC-16 (Cover) Dictamen final firmado por Responsable Sanitario.

4. Reglas de Oro de Xelle Scientific

  • Cadena de Frío: Rango mandatorio de 2°C a 8°C. Cualquier desviación invalida el lote inmediatamente.
  • Prevención de Congelación: Uso obligatorio de separadores físicos. El vial jamás debe tocar el gel congelado.
  • Trazabilidad: La resta entre fecha de ingreso y salida en bitácora jamás debe exceder los límites legales (NOM-087).
  • Identificación de Origen: Registrar siempre el área o proceso de donde emergió el residuo para detectar sobre-generación[cite: 4].
