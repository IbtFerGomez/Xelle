IT-OP-01 - Sistema Integral de Etiquetado

1. Semáforo de Estatus de Lotes (Identificación Física Principal)

Todo material en almacén o laboratorio debe portar una de las siguientes etiquetas según su fase:

Color Estatus Código Uso Normativo
Cuarentena
 PENDIENTE ET-LC-03 Material recién recibido o producto terminado esperando resultados de laboratorio (Fase 3).
Liberado
 APROBADO ET-LC-02 Producto que cumplió todas las especificaciones. Listo para distribución y uso clínico.
Rechazado
 NO CONFORME ET-LC-04 Material con fallas técnicas, contaminación o excursión térmica. Segregar para destrucción.
Muestreado
 CONTROL ET-LC-01 Sello de seguridad que indica que la caja fue abierta por Calidad para inspección.

2. Etiquetas de Materia Prima y Recepción

Código Nombre Descripción y Uso
ET-LC-05
(ET-MP-01)		 MP Placenta Identificación de materia prima - placenta. Se aplica en recepción logística incluyendo ID de donante, hospital origen, fecha/hora de recolección y recepción. Indica estatus inicial: Cuarentena o Aceptada.
ET-LC-06
(ET-MPNC-01)		 MPNC Placenta Materia Prima No Conforme - Rechazo de placenta. Indica motivo (logístico, macroscópico o biológico). Referencia manifiesto de destrucción (FO-LC-20). Papel ROJO.
ET-LC-08 Insumo Aprobado Control de inventarios para insumos aprobados. Incluye lote de proveedor, fecha de recepción/caducidad y requisitos de cadena de frío.

3. Etiquetas de Control de Calidad

Código Nombre Descripción y Uso
ET-LC-09
(ET-LC-12)		 Contramuestra Muestra de retención - Prohibida su venta. Se utiliza para retención de producto. Debe mantenerse Caducidad + 1 año. Estatus: Análisis CC o Retención Legal.
ET-LC-10
(ET-LC-13)		 Contenedor Retenciones Identificación de contenedor para custodia de muestras de retención en Control de Calidad. Indica rango de temperatura (2-8°C, -20°C, -80°C) y fecha de depuración.
ET-LC-11
(ET-LC-15)		 Muestra en Análisis Control de Calidad - No manipular. Identifica muestras bajo análisis (Esterilidad, Endotoxinas, Serología, etc.). Incluye fecha de inicio y analista responsable.

4. Etiquetas de Productos No Conformes

Código Nombre Descripción y Uso
ET-LC-07 PNC - Rechazo General Producto No Conforme (no placenta). Indica fase del rechazo (Manufactura/Expansión, Control Calidad, Devolución). Referencia reporte de desviación (FO-QA-03). Disposición: Destrucción o Retención CAPA.
ET-LC-06
(ET-MPNC-01)		 MPNC Placenta Específica para materia prima placenta no conforme. Ver sección 2.

5. Etiquetas de Productos Especiales y Subproductos

Código Nombre Descripción y Uso
ET-LC-12
(ET-LC-21-REV)		 Medio Condicionado Medio recuperado de cultivo celular. Trazabilidad de subproducto celular. Incluye ID lote origen, tipo celular, pasaje (P), volumen y condiciones de almacenamiento.
ET-LC-13
(ET-LC-22)		 Exosomas Recuperados Concentrado celular - Manejo en frío. Incluye método de aislamiento, concentración estimada y almacenamiento recomendado (-20°C o -80°C).
ET-LC-15
(ET-LC-24)		 Medio Liofilizado Insumo en polvo - Frágil. Control de humedad. Incluye peso neto, factor de reconstitución y advertencia de sello de vacío.

6. Gestión de Residuos (RPBI - NOM-087-SEMARNAT-SSA1-2002)

Código Color Contenido Autorizado
ET-RPBI-01 ROJO Punzocortantes, cultivos, cepas y residuos no anatómicos.
ET-RPBI-02 AMARILLO Placentas y residuos patológicos (tejidos anatómicos).

7. Flujo Documental y Puntos Críticos de Control (PCC)

Fase del Proceso Formato Obligatorio Etiquetas Asociadas Punto Crítico de Control (PCC)
1. Recepción FO-LC-17 ET-LC-05 (MP-01) Integridad de embalaje y temperatura (2°C a 8°C).
2. Procesamiento FO-LC-18 ET-LC-03 Peso neto, apariencia visual y asepsia de cabina.
3. Muestreo FO-LC-34 ET-LC-01, ET-LC-09 Toma estratificada y registro de cantidad volumétrica.
4. Calidad FO-LC-19 ET-LC-03, ET-LC-11 Serología, esterilidad y endotoxinas.
5. Liberación FO-LC-16 (Cover) ET-LC-02 Dictamen final firmado por Responsable Dirección.
6. Rechazo y segregación FO-LC-51, FO-LC-52 ET-LC-04, ET-LC-06, ET-LC-07 Identificación, generación registro de PNC/MPNC, segregación física y documental.
7. Retención/Contramuestra F-CAL-05 ET-LC-10, ET-LC-09 Custodia por Caducidad + 1 año, control de temperatura.
8. Gestión RPBI FO-LC-20 ET-RPBI-01, ET-RPBI-02 Segregación por tipo de residuo según NOM-087.

8. Resumen de Códigos de Etiquetas

  • ET-LC-01: Muestreado (Naranja) - Sello de Control de Calidad

  • ET-LC-04: Rechazado (Rojo) - No Conforme
  • ET-LC-05 (ET-MP-01): Materia Prima - Placenta
  • ET-LC-06 (ET-MPNC-01): Materia Prima No Conforme - Placenta
  • ET-LC-07: Producto No Conforme (General)
  • ET-LC-08: Insumo Aprobado - Inventarios
  • ET-LC-09 (ET-LC-12): Contramuestra - Retención
  • ET-LC-10 (ET-LC-13): Contenedor de Retenciones
  • ET-LC-11 (ET-LC-15): Muestra en Análisis
  • ET-LC-12 (ET-LC-21-REV): Medio Condicionado Recuperado
  • ET-LC-13 (ET-LC-22): Exosomas Recuperados
  • ET-LC-15 (ET-LC-24): Medio Liofilizado
  • ET-RPBI-01: RPBI Rojo - Punzocortantes y cultivos
  • ET-RPBI-02: RPBI Amarillo - Tejidos anatómicos
