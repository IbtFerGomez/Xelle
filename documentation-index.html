¿Cómo acceso al panel de administración? expand_more

1. Inicia sesión como Xelle_Fer (pass: 123)
2. Ve a Dashboard → Módulo "Administración"
3. Click "Panel de Administración Completo"
O directo: Abre admin-users.html

¿Qué documentación debo leer? expand_more

Para empezar: QUICK_START.md (5 min)
Para entender mejor: admin-guide.html (visual e interactivo)
Para detalles técnicos: ADMIN_SYSTEM_README.md
Para resumen visual: implementation-summary.html

¿Cuáles son los 4 roles disponibles? expand_more

1. ADMINISTRADOR - Acceso total (super_admin)
2. GESTOR CALIDAD - Calidad y documentación
3. USUARIO - Acceso básico
4. VENTAS - Módulos comerciales
